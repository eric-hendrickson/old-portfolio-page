About

I am a linguist/ESL teacher turned developer and a San Diego native. While living and working in China teaching English to Chinese and Korean university students, I discovered Linux, figured out how to use the command line, and taught myself basic scripting, and became eternally addicted to programming.

After returning to San Diego I spent the following years learning Java, C++, and other programming languages. I have just finished a coding bootcamp at LEARN Academy in downtown San Diego. My skills include Ruby on Rails, JavaScript (particularly jQuery, Node.js, Express, and Mithril.js), and Android development.

HidIt

HidIt is a Google Chrome Extension written in Javascript/jQuery that allows the user to turn virtually any web page into an interactive flash card. Simply click a word to cause it to disappear. Click it again to cause the word to reappear. If you need a little help, hover the mouse over the whitespace to see the word in a different color.

GitHub Repo: https://github.com/hendrickson02/HidIt

  • Date: November 2016
  • Setting: Commissioned Project
  • Category: Google Chrome Extension
